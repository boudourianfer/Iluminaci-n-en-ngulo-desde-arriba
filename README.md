# Iluminación-en ángulo-desde-arriba

Una iluminación con un ángulo acusado da luz Por lo tanto, resulta apropiada para fotografiar desnudos cuando se quieran destacar las formas de la modelo. 
Objetivo 
Distancia focal de 50 mm o superior. 
Exposición 
Se recomienda un exposímetro de luz incidente. Con exposímetro de luz reflejada ya esté incorporado a la cámara o no) se puede hacer una lectura desde muy cerca (unos diez cm) sobre la zona bien iluminada del sujeto, encima del pecho izquierdo, a la altura de la clavícula.
